# citecheck 사용법 (한글 빠른 안내)

> 원고의 인용/DOI를 Crossref에 대조해 **깨진 DOI · 제목/연도/저자 불일치 · 철회된 논문**을 잡아냅니다.

## 가장 쉬운 방법
이 폴더의 **`실행.command` 파일을 더블클릭** → 예시가 바로 돌아가고 사용법이 화면에 뜹니다.

## 터미널에서 (이미 설치돼 있음)
```bash
# 내 논문 참고문헌 검사
citecheck ~/Downloads/02_프로젝트/논문_투고/BELL_Paper3_Sleep_7th_Domain/references.bib

# 텍스트 참고문헌 목록도 가능
citecheck refs.txt --format text

# 정상 항목까지 모두 보기
citecheck refs.bib --verbose
```

## 결과 읽는 법
- `✓` 정상 — Crossref 기록과 일치
- `!` 경고 — 제목/연도/저자 불일치, 또는 DOI 없음 (확인 권장)
- `✗` 오류 — DOI가 Crossref에서 안 열림, 또는 **철회된 논문**

오류(`✗`)가 하나라도 있으면 종료코드 1 → 제출 전 점검 스크립트로도 쓸 수 있습니다.
