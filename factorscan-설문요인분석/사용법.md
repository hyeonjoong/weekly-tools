# factorscan 빠른 사용법

설문 척도가 **몇 개 요인으로 이루어졌는지, 요인분석에 적합한지, 각 문항이 어느 요인에 실리는지**
한 번에 뽑아 주는 도구입니다. (신뢰도 Cronbach α는 짝 도구 `surveyscan`이 담당)

## 가장 쉬운 방법 — 더블클릭

`실행.command` 파일을 **더블클릭**하세요. 번들 예시(수면 자가진단 8문항·80명)로
KMO·Bartlett·요인 수·적재량 표가 바로 출력됩니다.

> 처음 한 번은 macOS가 막을 수 있어요 → 파일 우클릭 → **열기** → **열기**.

## 내 데이터로 돌리기 (터미널)

CSV는 **행=응답자, 열=문항** 형식이면 됩니다.

```bash
# 1) 설정 파일로 (역문항·척도범위 지정 가능, 권장)
python3 -m factorscan.cli 내설문.csv --config 내설정.json

# 2) 문항 열을 직접 지정, ID 열 제외
python3 -m factorscan.cli 내설문.csv --items Q1,Q2,Q3,Q4,Q5,Q6 --id-col ID

# 3) 요인 수를 2개로 고정
python3 -m factorscan.cli 내설문.csv --id-col ID --n-factors 2
```

설정 파일(`내설정.json`) 예:
```json
{ "id_cols": ["ID"], "reverse": ["Q1","Q2"], "scale_range": [1,5] }
```

`pip install -e .` 를 한 번 해두면 `factorscan 내설문.csv ...` 로 더 짧게 쓸 수 있어요.

## 결과 읽는 법 (핵심만)

- **[1] 적합성**: `KMO ≥ 0.6`(≥0.8 우수) 이고 `Bartlett p < .05` 면 요인분석 OK.
- **[2] 요인 수**: `★` 표시(평행분석)와 `고유값>1`(Kaiser)이 몇 개인지 → 그만큼의 요인.
- **[3] 적재량**: 각 문항 행에서 별표(`*`, |0.40|↑)가 붙은 요인이 그 문항의 소속.
  두 요인에 모두 별표면 **교차적재**(애매한 문항).
- **[4] 점검 문항**: 공통성<0.3·문항-총점<0.3·MSA<0.5·교차적재 문항을 모아 알려줍니다.
  여기 뜨는 문항은 제거/수정을 검토하세요.

## 자주 쓰는 옵션

| 옵션 | 뜻 |
|---|---|
| `--n-factors K` | 요인 수 직접 지정 (기본: 자동/평행분석, 평행분석 끄면 Kaiser) |
| `--rotation none` | Varimax 회전 끄기 |
| `--reverse Q1,Q2 --scale-min 1 --scale-max 5` | 역문항 재점수화 |
| `--parallel-iter 0` | 평행분석 생략(더 빠름) |
| `--json` | JSON으로 출력(다른 분석에 넘길 때) |
