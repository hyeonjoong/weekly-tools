# surveyscan 빠른 사용법

설문 응답 CSV → **문항 기술통계 · 결측 · 역문항 · 하위척도 점수 · Cronbach α(신뢰도)** 리포트.

## 가장 쉬운 방법 (더블클릭)

`실행.command` 파일을 **더블클릭**하면 번들 예시(수면 설문 40명)로 분석 결과가 바로 뜹니다.
처음 한 번은 macOS가 막을 수 있어요 → 파일 우클릭 → "열기" → "열기".

## 내 데이터로 돌리기 (터미널)

CSV는 **행=응답자, 열=문항**, 첫 줄은 문항 이름. 빈 칸·`NA`는 결측으로 처리됩니다.

```bash
cd surveyscan-설문응답분석

# 1) 하위척도 설정과 함께 (권장) — config.json에 척도/역문항 정의
python3 -m surveyscan.cli 내설문.csv --config 내설정.json --id-col ID

# 2) 설정 없이 빠르게 — ID만 빼고 숫자 컬럼 전체를 한 척도로
python3 -m surveyscan.cli 내설문.csv --id-col ID

# 3) 결과를 파일로 저장
python3 -m surveyscan.cli 내설문.csv -c 내설정.json --id-col ID -o 결과.txt
```

설정 파일(`내설정.json`) 예시는 `examples/sleep_config.json` 을 복사해서 고치세요.
역문항(긍정문)은 `reverse_items`에 넣으면 자동으로 뒤집어 계산합니다. (이때 `scale_min`/`scale_max` 필수)

## 결과 읽는 법 (3줄 요약)

- **Cronbach α**: `.7` 이상이면 신뢰도 OK. (`.9 우수 / .8 양호 / .7 수용 / <.6 낮음`)
- **`← 제거시 α↑`** 표시가 뜬 문항: 빼는 게 신뢰도에 더 좋을 수 있음 → 검토.
- **하위척도 점수**: 역문항 반영 후 문항 평균. 논문 표·분석에 바로 사용.
