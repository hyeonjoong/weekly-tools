# 빠른 사용법 (paperforge)

보유 데이터 목록(JSON)을 넣으면 **멀티모달 논문 아이디어 표**를 만들어 줍니다.

## 가장 쉬운 방법
- 이 폴더의 **`실행.command` 더블클릭** → 예시 데이터로 아이디어 매트릭스가 바로 나옵니다.

## 터미널에서

```bash
# 1) 한 번만 설치
cd ~/Downloads/02_프로젝트/깃헙/paperforge-논문아이디어발굴기
python3 -m pip install -e .

# 2) 예시로 실행 (점수순 전체)
paperforge examples/sleep_moa_manifest.json

# 3) 상위 5개만 + 파일로 저장
paperforge examples/sleep_moa_manifest.json --top 5 --out ideas.md --csv ideas.csv
```

## 내 데이터로 쓰려면
`examples/sleep_moa_manifest.json`을 복사해 내 데이터셋으로 고치세요. 모달리티는
한글로 적어도 됩니다(`뇌파`, `워치`, `호흡밴드`, `설문`, `유저테스트`, `moa`).
`n`(표본수)을 적으면 "이 표본으로 가능한지"까지 자동으로 판정합니다.

```json
{
  "study": "내 연구",
  "datasets": [
    {"modality": "뇌파", "n": 35, "variables": ["alpha_power", "theta_power"]},
    {"modality": "설문", "n": 40, "variables": ["psqi_total", "stress_score"]}
  ]
}
```

## 결과 읽는 법
- **요약 매트릭스 표**: 위에서부터 추천 순위(멀티모달·실현가능·변수 풍부 순).
- **권장 N / 보유 N**: 보유 N이 권장 N보다 작으면 "표본 부족 우려" — 표본을 늘리거나
  더 큰 효과를 기대할 수 있는지 검토.
- **상세 블록**: 각 아이디어의 가설·변수·권장 통계분석·적합 저널·신규성 메모.

> 권장 N은 계획용 근사치입니다. 최종 검정력은 G*Power 등으로 확정하세요.
